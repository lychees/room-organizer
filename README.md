# 🏠 整理房间大作战 (Room Organizer)

一个用 [Three.js](https://threejs.org/) 构建的第三人称 3D 整理房间小游戏。房间里散落着书本、纸团，还有没摆好的椅子、乱糟糟的床铺和掉在地上的抱枕——把它们全部归置整齐吧！

## 🎮 在线试玩

**https://lychees.github.io/room-organizer/**

## 🕹️ 操作方式

| 按键 | 功能 |
| --- | --- |
| `W A S D` | 移动 |
| 鼠标 | 旋转视角（点击画面锁定指针） |
| `空格` | 跳跃 |
| `Shift` | 加速跑 |
| `E` | 交互（拾取 / 放置 / 开关） |
| `B` | 打开/关闭家具商店（模拟人生式购买模式） |

## 🛒 购买模式（类模拟人生）

按 `B` 打开家具商店，用模拟币 § 购买家具：

- 点击商品进入**放置模式**：半透明幽灵预览跟随鼠标（0.25m 网格吸附，绿色=可放置，红色=位置无效）
- **左键**放置（可连续放置）、**R** 旋转 90°、**右键/Esc** 取消
- 初始资金 §3000，每完成一项整理任务 **+§100**
- 金钱与已购家具通过 `localStorage` 自动存档，刷新不丢失（商店底部可清空存档）

## 🎯 游戏目标

完成全部 15 项整理任务：

- 📕 把散落各处的 8 本书捡起并放上**书柜**
- 🧻 把 4 个纸团扔进**垃圾桶**
- 🛏️ 整理床铺
- 🪑 把椅子放回书桌
- 🛋️ 把抱枕放回沙发

## 🪑 可交互家具

书柜、书桌抽屉、椅子、床、沙发抱枕、电视（开关）、落地灯（开关）、储物柜（开门）、盆栽（浇水）、垃圾桶等。

## 🚀 本地运行

纯静态页面，无需构建。任意静态服务器即可：

```bash
python -m http.server 8000
# 打开 http://localhost:8000
```

## 📁 项目结构

```
index.html          # 页面、HUD、商店面板
js/main.js          # 游戏主循环、角色控制、相机、交互系统、购买模式、存档
js/room.js          # 房间与基础家具建模、可交互物定义
js/shop.js          # 家具商店目录（电脑、弹珠台等）与音效
lib/three.module.js # Three.js r160（本地引用，无外部依赖）
```
