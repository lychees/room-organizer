# 🏠 整理房间大作战 (Room Organizer)

一个用 [Three.js](https://threejs.org/) 构建的第三人称 3D 整理房间小游戏。房间里散落着书本、纸团，还有没摆好的椅子、乱糟糟的床铺和掉在地上的抱枕——把它们全部归置整齐吧！

## 🎭 双时代模式

开局可选择两种时代风格（各自独立存档）：

- 🏙️ **现代公寓**：131 件现代家具，明亮的房间、方窗、电视、落地灯
- ⚓ **大航海 · 维多利亚**：62 件复古家具，深色木质房间、黄铜舷窗、石砌壁炉（可生火）、银烛台（可点燃）、船长礼服装扮的角色；商店里是船舵、海盗宝箱、留声机、油灯、航海图、六分仪、四柱床、翼背扶手椅、下午茶桌、打字机……

## 🕹️ 操作方式

| 按键 | 功能 |
| --- | --- |
| `W / S` | 前进 / 后退（相对角色朝向，与相机无关） |
| `A / D` | 左转 / 右转 |
| 鼠标 | 旋转视角（点击画面锁定指针） |
| `空格` | 跳跃 |
| `Shift` | 加速跑 |
| `E` | 交互（拾取 / 放置 / 开关） |
| `G` | 打开衣帽间换装 |
| `B` | 打开/关闭家具商店（模拟人生式购买模式） |

## 👔 换装系统

按 `G` 打开衣帽间，或对**衣帽架**按 `E`，即可给角色换装——服装选择会随存档保存。

- 🏙️ 现代（8 套）：蓝色休闲装、红色卫衣+棒球帽、工地工装+安全帽、黄色雨衣+雨帽、红白运动服+头巾、黑色西装、粉色睡衣+睡帽、绿色工装
- ⚓ 大航海·维多利亚（8 套）：船长礼服+礼帽、海盗装+红头巾、绅士燕尾服+高礼帽、水手服+水手帽、探险家装+探险帽、贵族紫袍+白色假发、蒸汽朋克装+飞行帽、学者长袍

## 🛒 购买模式（类模拟人生）

按 `B` 打开家具商店，用模拟币 § 购买家具：

- 点击商品进入**放置模式**：半透明幽灵预览跟随鼠标（0.25m 网格吸附，绿色=可放置，红色=位置无效）
- **左键**放置（可连续放置）、**R** 旋转 90°、**右键/Esc** 取消
- 初始资金 §3000，每完成一项整理任务 **+§100**
- 金钱与已购家具通过 `localStorage` 自动存档，刷新不丢失（商店底部可清空存档）

### 家具目录（131 件 / 14 个分类）

商店顶部有分类筛选：**桌椅、沙发、床、柜架、灯具、植物、电器、厨卫、娱乐、乐器、装饰、运动、收纳、宠物**。

特色交互家具举例：

- 💻 **电脑桌**：开机后屏幕跑代码雨
- 🎰 **弹珠台**：小球物理弹跳 + 音效 + 实时计分
- 🕹️ **街机**：像素 attract 动画，投币开玩
- 🎹 **钢琴 / 🎸 吉他 / 🥁 架子鼓 / 🎻 小提琴**：每种乐器音色旋律不同
- 🧊 **冰箱**：开门亮灯拿汽水；🌀 **洗衣机**：滚筒真的会转
- 🌬️ **电风扇**：摇头+叶片旋转；❄️ **空调**：导风板摆动
- 🐠 **鱼缸**：小鱼真的在游；🐱 **猫爬架** / 🐶 **狗窝** / 🐹 **仓鼠跑轮**
- 🎨 **画架**：每次互动真的在画布上添一笔；🕰️ **落地钟**：钟摆摇动 + 报真实时间
- 🥊 **拳击沙袋**：打了真的会晃；🏃 **跑步机**：跑带滚动；🚿 **淋浴房**：水流出现
- 📚 **小书柜 / 大书架**：散落的书放上去同样计入整理进度
- 还有台球桌、桌上足球、麻将桌、象棋桌、家庭影院、霓虹灯牌、云朵吊灯、樱花树、雕塑、地球仪……

地毯 / 瑜伽垫 / 吊灯 / 壁挂搁板为 **soft 家具**，可与其它家具重叠摆放（不挡路）。

## 🎯 游戏目标

完成全部 15 项整理任务：

- 📕 把散落各处的 8 本书捡起并放上**书柜**
- 🧻 把 4 个纸团扔进**垃圾桶**
- 🛏️ 整理床铺
- 🪑 把椅子放回书桌
- 🛋️ 把抱枕放回沙发

## 🪑 可交互家具

书柜、书桌抽屉、椅子、床、沙发抱枕、电视（开关）、落地灯（开关）、储物柜（开门）、盆栽（浇水）、垃圾桶等。

## 🚀 本地运行

纯静态页面，无需构建。任意静态服务器即可：

```bash
python -m http.server 8000
# 打开 http://localhost:8000
```

## 📁 项目结构

```
index.html          # 页面、HUD、商店面板
js/main.js          # 游戏主循环、角色控制、相机、交互系统、购买模式、存档
js/room.js          # 房间与基础家具建模、可交互物定义
js/shop.js          # 家具商店目录（电脑、弹珠台等）与音效
lib/three.module.js # Three.js r160（本地引用，无外部依赖）
```
